# Power BI Snooker Analytics

A Power BI and SQL analytics project analysing fictional professional snooker tournament data.

## Features
- Player performance analysis
- Tournament intensity heatmap
- Ranking vs age analysis
- SQL analytical queries
- Power BI dashboarding

## Tools Used
- Power BI
- SQL
- DAX
- Power Query
- Excel
